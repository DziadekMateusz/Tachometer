# Tachometer
Photoelectric tachometer, which operates on the principle of the photoelectric effect – a beam of infrared light emitted by a photodiode is reflected from a marker located on the rotating element being tested, from where it hits a phototransistor, which changes its resistance under the influence of the received light, so at the same time the analog readings on the microcontroller pin connected to the phototransistor change. By calculating the time that elapses between individual distinct changes in the readings, we can obtain the rotational speed of the object. Project designed and created during electronics lessons at university.
